# 연결점 목록

두가지 선택지  
1. `[]()` 방식은 미리보기가 안나온다. 대신 alt 텍스트와 함께 최대한 설명을 간결하게 해서 description 을 따로 넣던지.
2. 아니면 `[[]]` 위키링크 방식을 써야한다. 대신 파일명은 유일해야한다. index가 아닌 유니코드를 사용한다. 이러면 vscode에서의 이동이 조금 부자연스러워지지만 어차피 이 구조가 크게 변동이 없을거라 괜찮을것 같다.
  그리고 부가 설명을 넣으면 되지.
  아무래도 가장 좋은건 신뢰감을 주는 것이고 이 테마의 핵심이 다 보이는 건데 미리보기가 안되고 알수없는 문자가 뜨면 불안할 것 같다.
