위의 경로는 유니코드라서 안됐는데 (아예 url로 변환이 안되서 입력이 안됨, 빈문자열로 들어감) 일반 텍스트는 잘 이동이 된다. 확장자까지 입력을 하면 안되는건가. 참고로 한글도 인식이 안된다. 빈문자열로 되버린다. 영어만 가능.

일단 경로 넣을거면 기존 마크다운 방식으로 하고 위키링크로 넣을거면 파일명만 넣는게 좀더 직관적이긴 하다.
